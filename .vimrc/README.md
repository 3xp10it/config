### 0x01 Install[vim]

#### ubuntu & mostLinux
```
cd  
wget https://raw.githubusercontent.com/3xp10it/.vimrc/master/.vimrc_linux -O .vimrc
```

#### macOS
```
[macOS下优先选择用MacVim，不用vim]  
cd    
wget https://raw.githubusercontent.com/3xp10it/.vimrc/master/.vimrc_MacVim -O .vimrc
```
