### 0x01 Install[vim]

#### ubuntu & mostLinux
```
cd  
wget https://github.com/3xp10it/config/raw/master/.vimrc/.vimrc_linux -O .vimrc
```

#### macOS
```
[macOS下优先选择用MacVim，不用vim]  
cd    
wget https://github.com/3xp10it/config/raw/master/.vimrc/.vimrc_MacVim -O .vimrc
```
